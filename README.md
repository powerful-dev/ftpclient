<h2>⚙️ Installation</h2>

```bash
git clone https://github.com/yourusername/ftp-client.git

cd ftp-client

composer install

cp .env.example .env

php artisan key:generate
```

<h3>Run application</h3>

```bash
php artisan serve
```
